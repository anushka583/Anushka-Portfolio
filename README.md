# Anushka Sarkar — Portfolio

This portfolio was built as a deliberate engineering exercise; not just to showcase projects, but to design and ship a production-ready web application from scratch.

The goal was to create a scalable, modern, and maintainable frontend architecture that reflects how I approach development: structured, system-oriented, and detail-driven.

It represents:

- Component-based architecture with React
- Clean UI system using Tailwind CSS
- Responsive layout across devices
- Animated interactions and section transitions
- Working production deployment
- EmailJS integration for direct communication
- Environment variable handling
- Version-controlled and CI-ready structure

Rather than being a static resume replacement, this project is a continuously evolving platform. It serves as a sandbox for refining UI/UX decisions, improving frontend architecture, and experimenting with performance and interaction design.
